# Code Package for DRG: An Empirical Study of Dual-Reader Revision Gates for Quality and Compliance

## Description
This directory contains the experiment project used for the paper.

## Project Files
- `config.py`
- `data.py`
- `experiment_config.py`
- `experiment_harness.py`
- `main.py`
- `models.py`
- `training.py`

## How to Run
`python main.py`

## Dependencies
Install dependencies with `pip install -r requirements.txt` if needed.
